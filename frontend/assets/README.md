# Placeholder Assets

This directory contains placeholder images for the ROSAgo app.

## Required Assets

The following files should be added before deploying:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen (1284x2778px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px or larger)

For development, you can use simple solid color images or generate them using:
https://www.figma.com/community/plugin/735098390272716381/App-Icon-Generator
